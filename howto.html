> 08 — WEB AUDIO API (Sounds Without Files)

All sounds are generated mathematically — no MP3/WAV files needed:

let audioCtx = null; function playSound(freq, type, duration) { // Create audio context (only once) if (!audioCtx) audioCtx = new AudioContext(); // Oscillator = sound wave generator (like a tuning fork) const osc = audioCtx.createOscillator(); osc.type = type; // 'sine', 'square', 'sawtooth', 'triangle' osc.frequency.value = freq; // Pitch in Hertz // Gain = volume control const gain = audioCtx.createGain(); gain.gain.value = 0.1; // 10% volume // Connect: oscillator → volume → speakers osc.connect(gain); gain.connect(audioCtx.destination); // Play and auto-stop osc.start(); osc.stop(audioCtx.currentTime + duration); }

> 10 — CRYPTOGRAPHY ALGORITHMS

Caesar Cipher (ROT13)

// Shift each letter 13 positions. Apply twice = original text. function rot13(text) { return text.replace(/[a-zA-Z]/g, ch => { const base = ch <= 'Z' ? 65 : 97; return String.fromCharCode((ch.charCodeAt(0) - base + 13) % 26 + base); }); } // "HELLO" → "URYYB" "URYYB" → "HELLO"

Binary Encoding

// Convert each character to its 8-bit binary number: 'H'.charCodeAt(0) // → 72 (72).toString(2) // → "1001000" // Pad to 8 digits → "01001000" // "HI" → "01001000 01001001"

XOR Encryption

// XOR (^) = if bits differ, result is 1; if same, result is 0 // KEY PROPERTY: XOR with same key twice = original // data ^ key = encrypted // encrypted ^ key = data ← decrypted! const encrypted = text.split('').map((char, i) => { const keyByte = key.charCodeAt(i % key.length); return char.charCodeAt(0) ^ keyByte; });

Steganography (Zero-Width Characters)

// 4 invisible Unicode characters represent bit pairs: const ZW = [ '\u200B', // Zero-width space = 00 '\u200C', // Zero-width non-joiner = 01 '\u200D', // Zero-width joiner = 10 '\uFEFF', // Zero-width no-break = 11 ]; // 1. Convert secret to binary string // 2. Each 2-bit pair → one zero-width character // 3. Insert invisible chars inside normal text // Result looks normal but hides the message!

> 18 — STEGANOGRAPHY DEEP DIVE

Steganography hides data inside other data. Our Stego Lab hides text inside text using invisible Unicode characters that have zero visual width:

// Input: secret="hi", cover="Normal text" // Step 1: "hi" → binary: 01101000 01101001 // Step 2: split into 2-bit pairs: 01 10 10 00 01 10 10 01 // Step 3: each pair → invisible character: // 01 = \u200C 10 = \u200D 00 = \u200B etc. // Step 4: insert after first char of cover text // Output: "N[invisible chars]ormal text" // // It looks EXACTLY like "Normal text" on screen! // But pasting it into the decoder reveals "hi"

> 20 — FAQ
Is this real hacking? ▸
No. Everything is simulated. No networks scanned, no cameras accessed without permission, no files encrypted, no computers modified.
Is it safe for kids? ▸
Yes. All pranks are harmless. All scary screens reveal "just kidding". No data leaves the browser. The phishing quiz teaches real online safety skills.
Does it need internet? ▸
Only for Google Fonts on first visit. Everything else works 100% offline.
What's the password? ▸
anony (5 lowercase letters). After 3 wrong attempts, a bigger hint appears.
How do I exit an exploit? ▸
Press ESC or click/tap anywhere. Works for all fullscreen screens.
Where is my data stored? ▸
In your browser's localStorage. Never leaves your device. Clearing browser data resets everything.
Why no sound on first load? ▸
Browsers block auto-playing audio until a user interaction (click/tap). Once you click anything, all sounds work.
Does the webcam exploit access my real camera? ▸
Webcam Fake Hack: No — it shows animated static/noise, no real camera. Face Glitch: Yes — it requests camera permission (you can deny), applies distortion effects in real-time. The browser always shows a permission popup.
What's the fastest way to rank up? ▸
Konami Code (↑↑↓↓←→←→BA) = 500 XP instantly. Screen Takeover = 100 XP. Manifesto = 25 XP. Stack them!
Can I use this for a school project? ▸
Absolutely! Crypto Room teaches real cryptography. Slogans teach history. Password Tester teaches security. File Encryptor demonstrates XOR encryption. Terminal teaches command-line concepts.
How do I reset everything? ▸
Sidebar → Agent Profile → RESET_AGENT. For full reset: clear browser localStorage.
What are zero-width characters? ▸
Unicode characters that have no visual width — they're completely invisible. Used in real steganography to hide messages. Our Stego Lab uses \u200B, \u200C, \u200D, and \uFEFF to encode binary data inside normal-looking text.
Why vanilla JS and no React? ▸
Zero dependencies means: no npm install, no build step, no node_modules, no webpack config. Just open the HTML file. It works in 2024 and will work in 2034. React apps break when dependencies get outdated.
How does the Demo Mode work? ▸
It's an array of 39 async functions. A runner loop calls each function with await, showing a progress banner. Each function programmatically clicks buttons, types text, and opens sections — exactly like a user would. Abort cleans up all overlays.
How big is the codebase? ▸
5,683 lines across 5 files. 341KB total. 53 setup functions. 124 event listeners. 182 async/await calls. 61 distinct features. Zero external dependencies.
